# IVth Sem Lab Programs
A web application for accessing laboratory programs for fourth semester courses.

## Overview

This application provides a centralized platform for students to access laboratory programs for various courses including Design and Analysis of Algorithms (DAA), Microprocessors (MP), Database Management Systems (DBMS), and Unix Shell Programming.


## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install dependencies:
   ```
   npm install
   ```
4. Start the development server:
   ```
   npm run dev
   ```

## Usage

1. Open the application in your browser
2. Select a course from the homepage
3. Browse available lab programs for the selected course
4. View program code and sample outputs

## Project Structure

- `src/components/` - React components including Home, CoursePage, ProgramDetails, Header, and Footer
- `public/data/daa/` - Contains text files with actual program code for DAA course
- `src/App.css` - Main stylesheet with color scheme and layout definitions


## Deployment

Build the production version using:
